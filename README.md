# SnapSift
SnapSift is a lightweight, high-performance photo organization tool designed to turn cluttered image directories into structured libraries. Built for speed and precision, it allows users to rapidly preview, categorize, and move batches of photos with minimal friction.

🚀 Features
Lightning-Fast Sifting: Navigate through thousands of images with zero lag.

One-Key Sorting: Move or copy files to custom folders using keyboard shortcuts.

Composition Overlays: Integrated "Rule of Thirds" grid to help evaluate shots on the fly.

Controller Support: Map sorting actions to a gamepad for a more relaxed workflow.

Portable Design: Runs as a standalone utility directly from your local directory.

🛠️ Installation & Usage
Download: Clone this repository or download the latest release.

Locate Executable: Navigate to the dist folder.

Run: Launch SnapSift.exe (or the equivalent binary for your OS).

Organize: Select your source folder and start sifting.

💻 Tech Stack
Language: Python

GUI Framework: Tkinter / Custom Tkinter

Libraries: Pillow (PIL) for image processing
